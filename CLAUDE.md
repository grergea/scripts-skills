## 작업 규칙

### 스크립트 추가/수정 시

1. 스크립트 코드 수정 후 CLI 테스트로 동작 검증
2. 관련 스크립트 노트 동기화: `/Users/shlee/mynotes/03_Resources/Scripts/Skills/[스크립트명].md`
	- "스크립트 전문" 섹션 및 변경된 기능/사용법 섹션 업데이트, `updated` 날짜 갱신
3. 스크립트 추가/삭제 시 이 저장소의 `README.md`도 함께 갱신
4. 코드 + 노트 함께 커밋

